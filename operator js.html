Variables

There are three type of variables.

  1. Var
  2. Let
  3. Const

Var:

Var is a type of variable which can either be redeclared or reassigned

  • var a=10
  • a=20 // reassign
  • var a=30 // redeclared

Let:

Let is a type of variable which can only reassigned.

  • let a=10
  • a=20 // reassign
  • let a=30 // can not be redeclared

Const:

Var is a type of variable which can neithe be redeclared nor reassigned

  • var a=10
  • a=20 // neither reassign
  • var a=30 //nor redeclared

Two type Of Datatype

Primitive :

In primitive datatype only single value is stored.

Primitive are of five type

  1. Number
  2. String
  3. Boolean
  4. Undefiend
  5. Null

Non-Primitive :

In non-primitive datatype muliple value is stored.

Non-Primitive are of three type

  1. Array
  2. Object
  3. Function
